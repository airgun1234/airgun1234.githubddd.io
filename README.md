## 양자컴퓨터

### 양자컴퓨터란?

양자컴퓨터란 양자 얽힘이나 양자 중첩과 같은 양자역학적인 현상을 이용해 자료와 계산을 처리하는 컴퓨터이다. 고전적인 컴퓨터에서 자료의 양을 비트로 표현하는 것과 달리 양자컴퓨터는 큐비트를 이용해 자료를 표현하고 계산한다. 한 개의 처리 방식으로 여러 가지 계산을 동시에 할 수 있어서 정보처리량과 처리 속도가 기존의 컴퓨터와 비교 불가로 빠르다.
